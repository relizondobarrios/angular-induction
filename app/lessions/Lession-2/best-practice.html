Best practices

  • Nunca se asignan variables directamente al scope, se debe crear un modelo y a este modelo asignar dicha variable o propiedad. 
    • $scope.name = ‘raquel’;
$scope.user = { name: ‘raquel’ } ;
  • La inicialización de variables en el scope se hacen al inicio de los documentos.
  • Los archivos .css se agregan al inicio del documento layout y los archivos js se agregan al final del documento.
