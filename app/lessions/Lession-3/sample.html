induccion-angular app: one-time binding [5]

  • Vamos a crear un arreglo de objetos llamado products.
  • Esta variable la vamos asignar a un modelo llamado content.
  • Vamos hacer render de esta variable. Para esto vamos a crear un div que muestra la descripción de cada producto y el precio.
  • Ahora bien, vamos a agregar un input con el ngModel igual a cada precio de los productos. Podemos probar el two-way binding cambiando el valor del input.
  • Por último, agregamos :: al binding de la descripción y al binding del precio y volvemos a probar cambiando el valor en el input.
