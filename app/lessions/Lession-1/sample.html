induccion-angular app: descargar angular-seed [1]

  • Crearemos una carpeta donde vamos a guardar la aplicación.
    • induccion-angular
  • Abrimos una línea de comandos desde esta carpeta, y bajamos AngularJS seed mediante GIT.
    • git clone --depth=1 https://github.com/angular/angular-seed.git induccion-angular
  • Una vez clonado el repositorio, vamos a hacer:
    • cd induccion-angular npm install
  • Una vez que este proceso termine, haremos:
    • npm start

induccion-angular app: crear módulo [2]

  • Dentro de la carpeta induccion-angular vamos a encontrar una carpeta llamada ‘app’ en esta carpeta están todos los archivos relacionados con una aplicación Angular.
  • Para empezar un proyecto en Angular debemos crear un módulo que corresponderá al módulo principal de la aplicación.
    • var myApp = angular.module(“myApp”, []);
  • Ahora bien, definiendo este nombre para el módulo principal, vamos al HTML y en la etiqueta body agregamos:
    • < body ng-app=”myApp”>
